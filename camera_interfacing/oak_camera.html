OpenCV AI Kit (OAK) Camera

OAK-D Pro W has 2 black-and-white mono sensors on each side, 1 RGB sensor at the center, 1 IR Dot Projector (For Active Stereo), and 1 IR Illumination LED (For Low light environment). For using it with low-power (can’t supply more than 1 Amps through USB) Single Board Computer like Raspberry Pi, use the OAK Y Adapter for powering camera from dedicated power source.

Attention

All the examples are based on DepthAI v2 API. But, the concepts are similar and migration can be done with minimal changes based upon following Migration Guide to DepthAI v3 API.
